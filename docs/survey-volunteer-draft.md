# 志工供給端問卷草稿 v0.1

目的：驗證「社會人士遠距伴讀志工」的真實供給量與時段分佈（issue #1），並蒐集種子志工名單。
預計填答時間 3 分鐘。工具建議：SurveyCake 或 Google Forms。發放管道見文末。

判讀原則：「有興趣」不算數，**留 email + 勾具體時段 + 願意接受查核**三者齊備才算一個有效供給訊號。

---

## 開頭說明（放問卷首頁）

> 我們在研究一件事：讓下了班的大人，也能安全地遠距陪偏鄉或弱勢的孩子讀書、寫作業。
> 台灣的免費學習資源已經很多，缺的是「陪伴的人」。這份問卷想了解：像你這樣的人，願不願意、能不能成為其中之一。
> 全部 12 題約 3 分鐘。這是非營利的探索專案（github.com/ShihShihDer/bandu），資料僅用於需求評估。

## 第一部分：你是誰（粗粒度就好）

1. 年齡層：20 以下 / 21–30 / 31–40 / 41–50 / 51+
2. 目前身分：上班族 / 學生 / 自由工作者 / 家管 / 退休 / 其他
3. 是否有陪伴孩子的經驗（自己的小孩、家教、營隊、志工皆可）：有 / 沒有，但願意學

## 第二部分：意願與形式（核心題）

4. 如果有安全、有培訓的管道，你願意定期遠距陪一個孩子讀書嗎？
   非常願意 / 願意試試 / 看情況 / 應該不會（→ 跳到第 10 題）
5. 你實際上每週最多能穩定投入多少時間？（誠實就好，這題最重要）
   每週 1 小時 / 每週 2 小時 / 每週 3 小時以上 / 只能不定期
6. 你「真的有空」的時段是？（複選，勾實際會出現的，不是理想值）
   平日 12:00–13:30 / 平日 18:00–20:00 / 平日 20:00–22:00 / 週六白天 / 週日白天 / 週末晚上
7. 你比較能接受的陪伴形式？（複選）
   視訊 / 純語音 / 文字＋共用白板 / 都可以，看孩子需要
7-1. 如果內容不只是陪寫作業，而是**分享你的工作與興趣**（例如帶孩子看看工程師／護理師／廚師的一天），你的意願會：
   明顯提高 / 差不多 / 降低（我比較想單純陪課業）
8. 成為志工前，你願意接受哪些程序？（複選；全不勾者不列入有效供給）
   身分查核（由合作機構執行）/ 2–3 小時線上培訓 / 試教與配對面談 / 定期回報陪伴紀錄
9. 你願意承諾的最短陪伴週期？（陪伴最忌中途消失）
   一學期（約 4 個月）/ 半年 / 一年 / 無法承諾固定週期

## 第三部分：顧慮與動機

10. 讓你猶豫或卻步的原因是？（複選）
    怕自己教不好 / 時間不穩定 / 擔心安全與法律責任 / 不知道管道 / 對象孩子的狀況我不了解 / 其他____
11. 如果你參加，最主要的動機是？（單選）
    回饋社會 / 喜歡孩子 / 自己曾受過類似幫助 / 想累積教學經驗 / 其他____

## 第四部分：承諾訊號（真正的指標）

12. 這個計畫啟動時，願意收到通知嗎？
    願意，我的 email 是：____ / 暫時不用
13. （選填）願意參加一場 30 分鐘的線上說明會，聊聊你的想法嗎？願意 / 不用

## 發放管道（草案）

- PTT Volunteer 板、Dcard（心情/工作板帶公益脈絡發文）
- FB：公益/志工類社團、地方社團
- Threads / 個人社群（誠實寫「我在做一個公益專案的需求調查」）
- 目標樣本：先收 100 份；判讀門檻——有效供給訊號（見上）≥ 20 人且平日晚間時段集中，才代表「下班伴讀」模式有真實供給。
